Compatibility Module¶

Compatibility module. Provides implementations of classes or functions that are not available in earlier versions of Python.

author

Doug Skrypa

class requests_client.compat.cached_property(func)[source]¶

Bases: object

A decorator that converts a method into a lazy property. The wrapped method id called the first time to retrieve the result, and then that calculated result is used the next time the value is accessed. Deleting the attribute from the instance resets the cached value and will cause it to be re-computed.
